# Anime Library

This is a simple project I created with React + Vite

The purpose is to practice fetch() calls and connect with an API

Things Learned:
    - Practiced how to use fetch()
    - Introduced to the concept of debouncing (very useful!)
    - Practiced connecting components, using state, using props, how to access data from arrays
    - Learned more about how the proccess of the app rendering as state is updated.
